# 👟 Sneaker Monitoring Dashboard

A real-time web dashboard that monitors independent UK sneaker boutiques for new releases from **Nike, Vans, and New Balance**.

## 🚀 What it does
This application automates the "refresh" button. It scans the product feeds of top UK boutiques every 2 minutes and alerts you instantly when a new product matching your favorite brands is detected.

- **Live Log Feed**: Watch the script work in real-time.
- **Visual Alerts**: See product images, prices, and direct store links as they drop.
- **Smart Filtering**: Specifically targets the brands you care about.
- **Bot Protection Bypass**: Uses custom headers to mimic human browsing and avoid IP blocks.

## 🛠️ Tech Stack
- **Node.js & Express**: The engine running the server.
- **Socket.io**: Enables "Real-Time" communication (no page refreshing needed).
- **JavaScript (ES6)**: Core logic for data fetching and filtering.
- **CSS3**: Dark-mode dashboard interface.

## 📦 How to Run
1. **Clone the repository**
   ```bash
   git clone [https://github.com/your-username/sneaker-monitor.git](https://github.com/your-username/sneaker-monitor.git)
   cd sneaker-monitor
   ```
2. **Install Dependencies**
    ```bash
    npm install 
    ```
3. **Start the Monitor**
    ```bash
    node server.js
    ```
4. **View in Browser**
    Navigate to the designated localhost port and click "START SCANNER".
